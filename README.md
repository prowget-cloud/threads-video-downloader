# Threads Video Downloader

Extension sederhana untuk mendownload video dari Threads.net secara langsung melalui tombol yang disematkan di video.

## Fitur
- Auto-detect video di timeline.
- Injeksi tombol "Download" yang ringan.
- Tanpa iklan dan tanpa tracking.

## Cara Install (Manual)
1. Download atau clone repositori ini.
2. Buka Chrome dan pergi ke `chrome://extensions/`.
3. Aktifkan **Developer Mode** di pojok kanan atas.
4. Klik **Load unpacked** dan pilih folder `src` dari proyek ini.
5. Buka [Threads.net](https://www.threads.net) dan nikmati!
